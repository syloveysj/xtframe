## 技术架构
开发语言： Java

数 据 库 ： MySQL、SQLServer、Oracle

核心框架： Spring、SpringMVC、C3P0、Drools

缓存技术： Ehcache

全文检索： Lucene

模板视图： Freemarker、ExtJs4

权限控制： Shiro
